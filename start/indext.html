🔀 Git Push

What is git push?

It uploads your local commits to a remote repository (like GitHub, GitLab, or Bitbucket).

How it works

💻
Local Repo
(your computer)
➡️ git push
☁️
Remote Repo
(GitHub etc.)

Basic Syntax

git push <remote> <branch> git push origin main

Common Commands

Push to main branch:

git push origin main

Push a new branch for the first time:

git push -u origin my-branch

Force push (use carefully!):

git push --force

Typical Workflow

git add . git commit -m "your message" git push origin main
